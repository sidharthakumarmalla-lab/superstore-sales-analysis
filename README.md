# 📊 Superstore Sales Analysis

## 📌 Project Overview
This project analyzes Superstore sales data to identify trends, patterns, and business insights.

## 🎯 Objectives
- Analyze sales performance
- Identify top-performing regions and products
- Understand profit trends
- Provide actionable insights

## 📁 Dataset
- Source: Superstore dataset
- File: `data/superstore_sales.xlsx`

## 🛠 Tools Used
- Microsoft Excel
- Python

## 📊 Key Analysis
- Sales by region
- Profit analysis
- Category-wise performance
- Monthly trends

## 📈 Key Insights
- The West region generated the highest revenue
- Technology category showed the highest profit
- Some sub-categories are loss-making
